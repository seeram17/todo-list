#\ Todo List App

A clean and responsive **Todo List web application** built using **HTML, CSS, and vanilla JavaScript**.  
The app allows users to add tasks with optional dates, delete tasks, and interact using keyboard input.  
Designed with a modern **indigo/blue UI theme** and optimized for mobile devices.

---

##  Live Demo
 https://seeram17.github.io/todo-list/

---

##  Features
-  Add new todo items
-  Optional date support
-  Delete todos
-  Add todos using the Enter key
-  Fully responsive (mobile-friendly)
-  Modern indigo-themed UI
